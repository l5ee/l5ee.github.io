AE的30个高效快捷键

下面有超过30个 After Effects的快捷键列表，一定程度上能够提高我们的工作效率。

不想看文字可以看视频

图层属性

  • A（锚点），P（位置）S（比例）R（旋转）和 T（不透明度）

查看更多属性

  • 上述快捷键 A, P, S, R, T + Shift

快速设置关键帧

  • option + A，P，S，R，T (Mac)
  • Alt + Shift + A，P，S，R，T（Windows）

显示所有关键属性

  • 选择图层并按 U（也适用于多个图层）。

访问手形工具

  • 按住空格键并拖动。

时间轴缩放

  • 按 + 或 - 缩放时间轴。

缩放当前合成视图查看器

  • 只需使用 , 缩小，. 放大。

缩放当前视图到查看器

  • 点击 Shift + /，你的合成将匹配到查看器大小。

快速创建关键帧

  • 点按 F9

关键帧前后跳转

  • 使用 J 和 K 键 进行关键帧跳转。

快速跳转图层首尾

  • 通过使用 I 和 O 键 在图层首尾跳转。

在帧与帧直接移动

  • Page Down和 Page Up
  • 或者 Cmd + → 和 Cmd + ←

加快预览速度

  • 在数字键盘上按 Shift + 0。

查看所有操作属性

  • 双击 U 键。

快速将标记添加到时间轴

  • 要添加标记，请按数字键盘上的 * 键。
  • 或者按住 Shift + * 设置标记。

预合成

  • Cmd + Shift + C

轻松移动图层

  • 通过点击 [ 或 ] 将图层首尾移动到当前播放头位置

将播放头移到合成的开头或结尾

  • Home 或 End

缩放图层以适合当前合成

  • Cmd + Opt + F （长宽匹配当前合成大小）

切换标尺开/关

  • Cmd + R

切换不同的查看器分辨率

  • 完整：Cmd + J
  • 一半：Cmd + Shift + J
  • 四分之一：Cmd + Opt + Shift + J

替换图层

  • 保持图层选中 + 新图层拖动到选中图层

选择图层

  • Cmd +↓ 或 ↑

拆分图层

  • Cmd + Shift + D

显示图形编辑器

  • Shift + F3

搜索图层

  • Cmd + F

最大化任何面板

  • 悬停到窗口并按下 〜

隐藏或显示图层控件

  • Cmd + Shift + H

字符间距调整

  • option + ← 或 →

保存当前帧

  • Cmd + Opt + S

形状图层锚点居中

  • Opt + Cmd + Home

显示并隐藏网格

  • Cmd + '
