我的设计工作流

发表于 2018-04-24

这个主要说一下我是如何开展设计工作和管理设计相关文件的（这里主要是在 Mac 下的一套工作流程）,当然你可能有更好的方式,欢迎提出来交流,不喜勿喷…

素材哪里找

部分网站需要科学上网

  • 图片类免费途径(无版权)

    1. Unsplash: 不用说,高大上图片,质量又高,尺寸又大, 还提供API

    2. pexels: 不多说

    3. pixabay: 不多说

    4. moose: 这个主要背景简单,适合做素材,小清新的风格

什么Pinterest,dribbble和behance都不用多说了,那多数都是盗图,当然个人练习还是可以用用的

  • 图标 UI类
    1. icons8: 69,300 Free Flat Icons
    2. UI8: 大量付费素材, mockup, Kit和图标, 某宝很容易获取
    3. freebiesjedi: 免费的模板, PSD 源文件内容
    4. Sketh App Source: 话说3年多前刚刚接触 sketch 就是在这上面获得大量素材

Eagle - 管理素材

强烈案例 Eagle这个软件，支持Mac和Windows

浏览器插件支持，特别快速的收藏网页图片，支持的格式也超级多，包括几乎所有的图像软件的格式，和部分文档

IconJar - 管理图标

有一个自己的的 icon 素材库能让我们节省很多时间. 同时, 我们可以很快的将 UI 的框架搭建起来,而不是其抠细节.

这里我用到一个软件

这个软件很强大哦, 只需要吧你要保存的 icon拖入就行了, 而且整个软件将你收集的资源打包. 也就是你可以把这个打包文件放入任何同步盘(iCloud, 百度, Dropbox 等)中,就可以实现多设备管理资源.

同时可导出你想要的多种格式的 icon 文件, 也支持打包分享, 怎么样, 可以制作自己的图标库, 简直完美!!

RightFont, 字体管理

这里我使用RightFont来统一管理字体，这样可以快速检索检索到我要的字体，和快速安装卸载字体

我会在本地建立一个字体文件夹，做好分类，然后导入这个RightFont

同步素材

上面提到的Eagle，IconJar 和 RightFont 都支持各种同步云，你可以用下方任意工具来同步这是哪个工具的素材哦

我用的是iCloud来同步的

Abstract - UI设计稿管理

由于平常工作大多数是使用Sketch来进行设计，以往管理文件可能一个版本一个sketch文件，甚至一个版本多个sketch文件，然后经常还修改，管理文件苦不堪言。

不过现在我使用Abstract来管理，好处是什么呢，利用这个软件我可以回滚到任意时候我提交(commit)过的文件状态,

一个项目多个sketch文件，可多人协作，每个修改都做到有明细记录，支持在线标记留言。

再也不用担心文件怎么命名版本，怎么建立文件夹了。

难点在于这个软件利用的是Git的思想，可能很多设计师没有接触过，不过也很简单，看完这两篇文章，我相信你就会用了

完成上述所以操作，我们几乎可以实现在多个电脑设备，无缝进行设计工作，文件都是同步的，素材也是哦
