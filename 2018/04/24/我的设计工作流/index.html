素材哪里找

部分网站需要科学上网

  • 图片类免费途径(无版权)

    1. Unsplash: 不用说,高大上图片,质量又高,尺寸又大, 还提供API

    2. pexels: 不多说

    3. pixabay: 不多说

    4. moose: 这个主要背景简单,适合做素材,小清新的风格

什么Pinterest,dribbble和behance都不用多说了,那多数都是盗图,当然个人练习还是可以用用的

  • 图标 UI类
    1. icons8: 69,300 Free Flat Icons
    2. UI8: 大量付费素材, mockup, Kit和图标, 某宝很容易获取
    3. freebiesjedi: 免费的模板, PSD 源文件内容
    4. Sketh App Source: 话说3年多前刚刚接触 sketch 就是在这上面获得大量素材

如何管理 icon

这里主要是在 Mac 下的一套工作流程

有一个自己的的 icon 素材库能让我们节省很多时间. 同时, 我们可以很快的将 UI 的框架搭建起来,而不是其抠细节.

这里我用到一个软件.

这个软件很强大哦, 只需要吧你要保存的 icon拖入就行了, 而且整个软件将你收集的资源打包. 也就是你可以把这个打包文件放入任何同步盘(iCloud, 百度, Dropbox 等)中,就可以实现多设备管理资源.

同时可导出你想要的多种格式的 icon 文件, 也支持打包分享, 怎么样, 可以制作自己的图标库, 简直完美!!
